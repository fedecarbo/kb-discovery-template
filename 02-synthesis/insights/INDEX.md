## How to read this

Each row: `**I-00X** (status, N sources, last YYYY-MM-DD): claim headline — optional flag`.

- **Status** follows the disconfirmation discipline: `hunch` → `emerging` → `strong` → `contested`. See the README ("Insights graduate by disconfirmation, not just accumulation") and `02-synthesis/probes.md` for the promotion rules.
- **N sources** = supporting evidence rows in the insight's table.
- **Flags** call out claims that are mid-flight: under tension, sample-biased, survivorship-bias-risk, etc.

If two consecutive sessions support a claim but both come through the same channel, the *sample is selecting for the evidence* — the source count understates the bias. Read the flag carefully before treating any insight as ready to promote.
